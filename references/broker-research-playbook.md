# 券商研报操作手册

当用户要求卖方研究、券商研报、银行研究、策略报告、行业报告、宏观报告或分析师研究时，使用此文件。

## 目标

在保持来源合法性和访问标注准确性的前提下，找到研究报告的最高质量可访问版本。

## 什么算好的结果

按优先顺序：
1. 机构官方网站上的公开 PDF (`official PDF`)
2. 包含标题、研究部门或分析师、日期和摘要的官方落地页 (`official page`)
3. 明确标注机构信息的授权门户或活动页面
4. 仅当没有更好的公开来源时才使用转载镜像 (`secondary mirror`)

## 券商研报类型

### 公开且通常可获取

- 年度展望合集
- 行业展望
- 主题策略报告
- 宏观展望
- 会议演示文稿
- 入门材料（primer）
- 公开洞察和思想领导力 PDF

### 通常受限

- 公司首次覆盖报告（initiation）
- 业绩前瞻或回顾
- 估值更新
- 仅订阅用户可见的行业报告
- 分析师门户下载内容

### 高风险来源

- 无来源追溯的文件托管站镜像
- 论坛或博客转载
- 被剥离出版方信息的 PDF

## 搜索模式

### 1. 确认机构

规范化以下信息：
- 机构英文名称
- 本地语言品牌名称（如相关）
- 研究部门名称（如独立）

### 2. 优先搜索官方公开平台

查找：
- 洞察页面
- 研究页面
- 思想领导力栏目
- 会议或活动页面
- 年度展望合集页面

### 3. 按文档类别搜索

将机构名称与以下关键词组合：
- `outlook`
- `strategy`
- `sector`
- `macro`
- `primer`
- `conference`
- `pdf`
- `研报`
- `策略`
- `行业`
- `宏观`

### 4. 验证访问状态

分配以下标签之一：
- `official PDF`
- `official page`
- `licensed/paywalled`
- `secondary mirror`

不要模糊这些类别之间的界限。

## 输出字段

对于每个来源，记录：
- 机构
- 文档标题
- 分析师、研究部门或研究团队（如可见）
- 出版日期
- 主题或行业
- 访问标签
- 官方页面 URL
- 直接 PDF URL（如可用）

## 决策规则

### 当仅存在受限的官方页面时

- 保留该来源
- 标注 `licensed/paywalled`
- 根据可见的元数据摘要说明其相关性
- 避免用可疑的转载替代，仿佛二者等价

### 当仅存在镜像 PDF 时

- 仅在该主题的其他路径完全受阻且用户重视完整性超过纯净性时才保留
- 标注 `secondary mirror`
- 说明未找到原始公开版本

### 当出现大量近似重复的报告时

优先选择：
1. 机构旗舰报告优于短篇简报
2. 用户时间窗口内最新版本
3. 更广泛的行业或主题报告优于公司维护性报告，除非用户明确要求公司覆盖

## 中国特定策略

积极使用中文机构名称和文档类别。

常用检索词：
- `研究所`
- `行业深度`
- `策略年度`
- `晨会纪要`
- `专题报告`
- `深度报告`

默认规则：
- 优先选择官方研究所或券商洞察页面
- 如果市场主要暴露报告元数据而非 PDF，则先返回元数据和官方路径

## 全球银行策略

在进行广泛搜索之前，先使用以下类别：
- 投行洞察平台
- 全球研究展望页面
- 会议演示页面
- 公开政策或市场评论页面

如果机构将营销洞察与持牌研究分开：
- 两者相关时均保留
- 将营销洞察视为公开支持材料
- 将持牌研究标注为 `licensed/paywalled`

## 禁止事项

- 暗示转载是经授权的
- 省略访问限制说明
- 在官方页面存在的情况下使用第三方摘要
- 混淆公开思想领导力内容与分析师研究而不加以区分标注
