# 来源优先级

除非用户给出更严格的指示，否则按以下顺序排列。

## 优先级阶梯

1. `official PDF`
   由出版机构、发行人、监管机构、交易所、公司、协会或授权活动页面直接托管的 PDF。
2. `official page`
   含有报告、演示文稿、会议记录或下载按钮的官方落地页，即使 PDF 本身不可用。
3. `authorized distribution`
   公开可访问的数据库、活动微型网站或合作伙伴页面，明确标注原始出版方和文档信息。
4. `secondary mirror`
   转载、聚合站或文件托管平台。仅在无法公开找到原始来源时使用，并标注风险。
5. `commentary`
   引用该报告的新闻报道、博客、演示文稿或 SEO 摘要。仅用于发现线索或补充背景，不作为主要交付物。

## 首选来源类别

### 1. 咨询和专业服务研究 (consulting report)

优先选择来自原始机构的命名研究中心、洞察页面和可下载报告。

典型目标：
- McKinsey
- BCG
- Bain
- Deloitte
- PwC
- EY
- Kearney
- Oliver Wyman
- Roland Berger
- Accenture
- Capgemini

### 2. 监管机构、交易所和公共机构 (regulator/exchange)

当用户需要市场结构、政策、合规或官方统计数据时使用。

典型目标：
- 证券监管机构
- 证券交易所
- 部委和政府机构
- 多边机构
- 中央银行

### 3. 公司投资者关系与年度出版物 (company IR)

用于战略分析、市场规模线索、产品路线图、年报、投资者演示文稿、资本市场日材料和可持续发展报告。

典型目标：
- 投资者关系网站
- 年报门户
- 业绩演示资料库
- 投资者日页面

### 4. 标准机构、行业协会和产业联盟 (association/standard)

用于技术白皮书、标准、基准框架和全行业采用数据。

典型目标：
- 标准化组织
- 行业协会
- 提供可下载论文的会议论文集
- 协会研究中心

### 5. 券商和银行研究 (broker research)

此类来源需谨慎处理，因为可获取性因许可不同而异。

按优先顺序：
1. 机构官方网站上发布的公开研究、展望、策略报告和主题报告。
2. 官方域名上的公开会议演示、行业入门材料、年度展望合集和宏观策略 PDF。
3. 持牌门户或仅客户可访问的页面，明确标注 `licensed/paywalled`。
4. 非官方转载仅作为最后手段，明确标注 `secondary mirror`，绝不将其描述为持牌或官方来源。

当用户要求券商研报时：
- 首先搜索机构官方页面
- 保留原始标题、分析师或研究部门名称（如有）及日期
- 明确说明访问限制
- 优先提供元数据加官方页面，而非可疑的转载链接

## 排名规则

以下情况得分更高：
- 更接近原始出版方
- 在相关时间窗口内更新
- 直接涉及请求主题而非相邻话题
- 可下载为 PDF 或演示文稿
- 有明确日期和可追溯的作者/机构

以下情况得分更低：
- 未标注日期
- 被第三方大量摘要
- 跨多个镜像站重复且无来源追溯
- 以 SEO 优化为目的而非出版质量

## 平分决胜规则

当两个来源看起来相似时，优先选择：
1. 原始托管站优于转载
2. PDF 优于 HTML 摘要
3. 有明确署名机构优于通用门户
4. 精确出版日期优于模糊的仅年份标注
5. 综合旗舰报告优于简短的衍生摘要
