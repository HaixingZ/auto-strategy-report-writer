# Workflow

## Fixed sequence

1. Compile brief
2. Produce Storyline Packet
3. Plan modules
4. Draft baseline report
5. Review and score
6. Rewrite highest-leverage weaknesses
7. Check plateau
8. Finalize

## Brief checklist

- Topic
- Purpose
- Audience
- Time scope
- Geography
- Object definition
- Must-answer questions
- Known materials
- Style rules
- Forbidden patterns
- Boundaries

Ask only for missing information that would materially change the report.

## Baseline rule

The first integrated draft is a baseline, not a final answer.
It should be reviewable and structurally complete, even if some evidence gaps remain.

## Rewrite rule

Each round may target only 1-2 high-leverage weaknesses.
Do not do full rewrites unless the thesis or structure is wrong.

## Plateau rule

Stop when:
- core quality dimensions are acceptable
- the last 2-3 iterations show minimal score gain
- remaining improvement depends mainly on human-provided input or interviews

## Human-input pause rule

If the strongest blocker is missing input, pause optimization and output:
- current best draft
- Evidence Gap Log
- Next-input Request
